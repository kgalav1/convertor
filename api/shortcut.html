Shortcut for SanGST [Works with Mozilla Firefox(V58 and Upper)]

  • F1 - Toggle Search Box in Reports
  • F2 - Search in Reports
  • CNTRL+SHIFT+F - Search in Menu
  • ALT+ENTER - Submit form except Master
  • ALT+H - HOME
  • ALT+S - SALE
  • ALT+P - PURCHASE
  • ALT+C - CHALLAN
  • ALT+X - VOUCHER ENTRY
  • ALT+E - EXPENSE ENTRY
  • ALT+SHIFT+S - GST SALE REPORT
  • ALT+SHIFT+P - GST PURCHASE REPORT
  • ALT+SHIFT+I - ITEM REPORT
  • ALT+SHIFT+J - ITEM STOCK REPORT
  • ALT+SHIFT+K - INVOICE REPORT
  • ALT+SHIFT+L - LEDGER REPORT
  • ALT+SHIFT+T - TRIAL BALANCE REPORT
  • ALT+SHIFT+V - PROFIT LOSS REPORT
  • ALT+SHIFT+B - BALANCE SHEET REPORT
  • ALT+SHIFT+X - LOGOUT
