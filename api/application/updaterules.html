WRITE Command

Use **  between two command to seprate the command.
Command 1
**
Command 2

Example:
ADD_CLM|comp_datatable|ref_id|VARCHAR|20|1|NULL|tel_no
**
ADD_CLM|comp_datalist|data_item|INT|1|1|0|final_rate
**
ADD_CLM|comp_datalist_##FY##|data_item|INT|1|1|0|final_rate

CHANGE/CREATE Financial Year Table

Use _##FY## after the table name to update in financial year table.
table_name_##FY##

Example:

If Change in both the table write command like below:
ADD_CLM|comp_datalist|data_item|INT|1|1|0|final_rate
**
ADD_CLM|comp_datalist_##FY##|data_item|INT|1|1|0|final_rate

To crate both the table master and financial year table (if avilable):

	`name` VARCHAR(100) NULL DEFAULT NULL,

ADD Column

ADD_CLM|table_name|column_name|type|constraint|allowNull(Y for NULL, N for NOT NULL)|default(If no default value use NULL)|after

Example:
ADD_CLM|comp_datalist|data_item|INT|1|Y|0|final_rate
ADD_CLM|comp_datalist_##FY##|data_item|INT|1|Y|NULL|final_rate

ALTER Column

ALT_CLM|table_name|old_column_name|new_column_name(If no change in name use old column name)|type|constraint|allowNull(Y for NULL, N for NOT NULL)|default(If no default value use NULL)|after

Example:
ALT_CLM|comp_item_master|data_item|data_item|INT|11|N|XYZ|bar_code
ALT_CLM|comp_item_master|data_item|data_val|INT|11|N|XYZ|bar_code

DROP Column

DRP_CLM|table_name|column_name

Example:
DRP_CLM|comp_data|md_date

CREATE Table

CRT_TBL|table_name|add_column_command_comma_seprated

Example:
CRT_TBL|comp_data|
	`id` INT(11) NOT NULL AUTO_INCREMENT,
	`name` VARCHAR(100) NULL DEFAULT NULL,
	`behaviour` VARCHAR(50) NULL DEFAULT NULL,
	`type` INT(11) NULL DEFAULT NULL,
	PRIMARY KEY (`id`)

RENAME Table

RNM_TBL|old_table_name|new_table_name

Example:
RNM_TBL|comp_data|comp_metadata
RNM_TBL|comp_data_##FY##|comp_metadata_##FY##

DROP Table

DRP_TBL|table_name

Example:
DRP_CLM|comp_data
DRP_CLM|comp_data_##FY##

Run QUERY Direct

QRY|query

Example:
QUERY|insert into ##DB##.table (col1,col2) values (1,'AA')
QUERY|insert into ##DB##.table_##FY## (col1,col2) values (1,'AA')
