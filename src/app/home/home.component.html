Image Types
  • SVG: Scalable Vector Graphics - Vector-based image format that supports smooth lines and curves.
  • TIFF: Tagged Image File Format - Versatile format for high-quality images with rich color depth.
  • ICO: Icon Image - File format for storing icon images primarily associated with Windows applications.
  • PSD: Adobe Photoshop Document - Native file format of Adobe Photoshop for advanced image editing.
  • AI: Adobe Illustrator Document - Native file format of Adobe Illustrator for scalable vector graphics.
  • PDF: Portable Document Format - Document format that can contain images and be converted to images.
  • EPS: Encapsulated PostScript - File format for vector graphics and high-resolution images.
  • TGA: Truevision Graphics Adapter - Format for storing high-quality images and animations.
  • HDR: High Dynamic Range Image - Format for capturing and displaying a wide range of luminosity levels.
  • EXR: OpenEXR Image - High-dynamic-range image format for storing and processing HDR images.
  • PNM: Portable Any Map - File format for storing images in various color spaces and bit depths.
  • WBMP: Wireless Bitmap Image - Bitmap image format for mobile devices.
  • XPM: X Pixmap - Format for storing color icons and small images.
  • J2K/JPEG2000: JPEG 2000 Image - Image format with superior compression and image quality.
  • HEIC/HEIF: High-Efficiency Image Format - Modern image format that provides high compression and quality.
